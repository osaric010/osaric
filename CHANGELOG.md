## التثبيت السريع

```bash
# استنساخ المشروع
git clone <repository-url>
cd osaric-accounts

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل الهجرات
python manage.py migrate

# إنشاء مستخدم إداري
python manage.py createsuperuser

# إنشاء البيانات التجريبية
python create_sample_data.py

# تشغيل الخادم
python manage.py runserver
```
